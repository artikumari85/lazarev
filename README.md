# LAZAREV. — Digital Product Design Agency

A responsive frontend recreation inspired by the LAZAREV. agency website. The project highlights a modern AI/product-design agency layout with animated navigation, case-study sections, video interactions, service accordions, and a responsive footer.

## Preview

Open `index.html` in a browser, or use the VS Code **Live Server** extension.

```text
http://127.0.0.1:5500/index.html
```

## Features

- Responsive layouts for desktop, tablet, and mobile
- Animated navigation menu and mobile hamburger menu
- Hero section with moving client logos
- Interactive case-study cards and hover video previews
- Showreel video interaction
- Expandable UI/UX and product-design service sections
- Digital product design process section
- Closing CTA and responsive footer

## Built With

- HTML5
- CSS3
- JavaScript
- [GSAP](https://gsap.com/) and ScrollTrigger
- [Locomotive Scroll](https://github.com/locomotivemtl/locomotive-scroll)
- [Remix Icon](https://remixicon.com/)

## Project Structure

```text
├── index.html     # Page structure and content
├── style.css      # Styling, animations, and responsive breakpoints
├── script.js      # GSAP, menu, scroll, and video interactions
└── README.md      # Project documentation
```

## Run Locally

1. Clone the repository.
2. Open the project folder in VS Code.
3. Start `index.html` using Live Server, or open it directly in a browser.

## Notes

Some images, videos, fonts, and animation libraries are loaded from public CDNs. An internet connection is required for those external assets to display correctly.
