# LAZAREV. — Agency Website Recreation

A responsive frontend recreation of the LAZAREV. digital product design agency website. It features a modern agency layout, smooth scrolling, GSAP interactions, video previews, expandable service sections, and a mobile navigation menu.

## Live preview

Open `index.html` directly in a browser or run it with the VS Code Live Server extension.

```text
http://127.0.0.1:5500/index.html
```

## Features

- Responsive desktop, tablet, and mobile layouts
- Desktop dropdown navigation and mobile hamburger menu
- GSAP loading, hover, and scroll-driven animations
- Locomotive Scroll smooth-scrolling setup
- Interactive showreel and project-video previews
- Expandable UI/UX and product design service sections
- Responsive process and footer sections

## Tech stack

- HTML5
- CSS3
- JavaScript
- [GSAP](https://gsap.com/) with ScrollTrigger
- [Locomotive Scroll](https://github.com/locomotivemtl/locomotive-scroll)
- [Remix Icon](https://remixicon.com/)

## Project structure

```text
├── index.html   # Page markup
├── style.css    # Styling and responsive rules
├── script.js    # Scroll, animation, video, and menu logic
└── README.md    # Project documentation
```

## Run locally

1. Clone the repository.
2. Open the folder in VS Code.
3. Start `index.html` with Live Server, or open it in a browser.

## Notes

Images, videos, fonts, and animation libraries are loaded from public external sources, so an internet connection is required for all visual assets to appear.
